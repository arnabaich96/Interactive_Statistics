Interactive Height Analysis: Step-by-Step Guide

Introduction

This interactive document guides you through analyzing the heights of individuals, step by step. You will start by generating the data, then proceed to select the type of summary statistics or visualizations you would like to see.

Step 1: Data Generation

Please input the number of individuals and the maximum height (in cm) for the data generation.

data_gen_ui <- fluidPage(
  numericInput("n", "Number of individuals:", value = 100, min = 1),
  numericInput("max_value", "Maximum height (cm):", value = 200, min = 50),
  actionButton("generate", "Generate Data"),
  tableOutput("heights_table")
)

data_gen_server <- function(input, output, session) {
  heights <- eventReactive(input$generate, {
    runif(input$n, min = 50, max = input$max_value)
  })
  
  output$heights_table <- renderTable({
    req(heights())
    head(heights(), 10)
  })
  
  return(heights)
}

heights <- shinyApp(ui = data_gen_ui, server = data_gen_server)

Step 2: Choose Summary Statistic or Plot

Now that the data has been generated, please select the type of summary or plot you would like to see.

summary_choice_ui <- fluidPage(
  selectInput("summary_choice", "Choose a summary statistic or plot:",
              choices = list(
                "Five-Number Summary" = "five_num",
                "Histogram" = "histogram",
                "Boxplot" = "boxplot",
                "Density Plot" = "density",
                "Detailed Summary Measures" = "detailed"
              )),
  actionButton("show_summary", "Show Summary/Plot"),
  uiOutput("summary_ui")
)

summary_choice_server <- function(input, output, session, heights) {
  
  observeEvent(input$show_summary, {
    choice <- input$summary_choice
    
    output$summary_ui <- renderUI({
      if (choice == "five_num") {
        verbatimTextOutput("five_num_summary")
      } else if (choice == "histogram") {
        plotOutput("histPlot")
      } else if (choice == "boxplot") {
        plotOutput("boxPlot")
      } else if (choice == "density") {
        plotOutput("densityPlot")
      } else if (choice == "detailed") {
        verbatimTextOutput("detailed_summary")
      }
    })
  })
  
  output$five_num_summary <- renderPrint({
    req(heights)
    summary(heights())
  })
  
  output$histPlot <- renderPlot({
    req(heights)
    hist(heights(), 
         main = "Histogram of Heights", 
         xlab = "Height (cm)", 
         ylab = "Frequency", 
         col = "lightblue", 
         border = "black")
  })
  
  output$boxPlot <- renderPlot({
    req(heights)
    boxplot(heights(), 
            main = "Boxplot of Heights", 
            ylab = "Height (cm)", 
            col = "lightgreen", 
            border = "black")
  })
  
  output$densityPlot <- renderPlot({
    req(heights)
    plot(density(heights()), 
         main = "Density Plot of Heights", 
         xlab = "Height (cm)", 
         ylab = "Density", 
         col = "blue")
  })
  
  output$detailed_summary <- renderPrint({
    req(heights)
    summary_stats <- list(
      Mean = mean(heights()),
      Median = median(heights()),
      StdDev = sd(heights()),
      Min = min(heights()),
      Max = max(heights()),
      Quantiles = quantile(heights(), probs = c(0.25, 0.5, 0.75))
    )
    summary_stats
  })
}

summaryApp <- function(heights) {
  shinyApp(ui = summary_choice_ui, server = function(input, output, session) {
    summary_choice_server(input, output, session, heights)
  })
}

summaryApp(heights)

Step 3: View Summary or Plot

After selecting the desired summary statistic or plot, it will be displayed here with an explanation.

  • Five-Number Summary: Includes the minimum, first quartile, median, third quartile, and maximum.
  • Histogram: Visual representation of the distribution of heights.
  • Boxplot: Summary of the distribution, showing the median, quartiles, and potential outliers.
  • Density Plot: A smoothed version of the histogram, showing the probability density of heights.
  • Detailed Summary Measures: Includes mean, median, standard deviation, and other statistics.

Conclusion

This interactive analysis has guided you through generating data, selecting the type of summary or visualization, and displaying the results. This structured approach helps in understanding the data distribution and key statistics in a clear and interactive manner.
